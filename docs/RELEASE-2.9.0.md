# Aulama Anime TV v2.9.0

## 更新內容

- 重整首頁、發現、時間表與作品詳情介面，改善電視距離下的資訊層級與焦點狀態。
- 長作品標題只在超出可視範圍時平滑循環，並加入兩側漸隱銜接。
- 加入 Claude 風格粗體標題字體與封面主題色。
- 修復遙控器在播放器控制列隱藏後失去操作焦點的問題。
- 修復一般播放偶發黑畫面，並保留失敗時的安全回退。
- 新增可選的「4K 快速」Lanczos 升頻模式；預設關閉，效能不足或掉幀時自動回退。
- 更新 Android TV 橫幅及自適應圖示，避免圓形與卡片裁切。

## 安裝

可直接覆蓋安裝舊版本；應用程式內的「檢查更新」亦會讀取最新 GitHub Release。
